# TypeScript Vitest boilerplate

This boilerplate uses TypeScript and Vitest as testing framework.

## Installing dependencies

```shell
# With node
npm ci
# With https://bun.sh
bun i
## With https://pnpm.io
pnpm i
## With https://yarnpkg.com/
yarn
```

## Running tests

For other packagers replace `npm` with `bun` | `pnpm` | `yarn`.

```shell
# Watch tests
npm test

# Run tests once
npm run test:once
```
